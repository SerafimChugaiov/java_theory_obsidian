1. **Top level class** (обычный класс - это основной тип класса, который может содержать поля, методы и конструкторы, и может быть создан вне других классов):
	- **abstract class** (Абстрактные классы - класс не может быть инстанцирован напрямую и служит основой для других классов, определяя абстрактные методы, которые должны быть реализованы в подклассах. [[Абстрактные классы|Подробнее...]]); 
	- **final class** (финализированный класс - не может быть унаследован, что обеспечивает его неизменность в иерархии наследования);
2. **Interfaces** (**Интерфейсы Java** - определяет набор абстрактных методов, которые класс обязан реализовать, и используется для описания поведения без реализации. [[Интерфейсы Java|Подробнее...]]) 
3. **Enum** (Enum Перечисления Java - это специальный тип класса, который представляет собой фиксированный набор констант. [[Enum Перечисления Java|Подробнее...]]);
